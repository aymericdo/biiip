# Biip landing page

Landing page React/Vite pour Biip, la petite boîte qui permet d'ouvrir son portail depuis un téléphone en actionnant physiquement la télécommande existante.

## Développement

```bash
npm install
npm run dev
```

Copiez `.env.example` vers `.env.local` et renseignez `DISCORD_WEBHOOK_URL` pour recevoir les demandes de contact sur Discord.
